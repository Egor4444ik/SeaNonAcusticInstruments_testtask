# Программа прореживания последовательностей отсчётов

## 1. Назначение программы

Программа предназначена для выполнения операции **прореживания последовательности отсчётов**, представленных в виде пар целых чисел *(X, Y)*, упорядоченных по значению **X**.

Прореживание выполняется по следующему принципу:

- последовательно анализируются *блоки* элементов, в которых значения **Y** одинаковы и следуют подряд;
- для каждого блока сохраняются:
  - первый элемент блока;
  - последний элемент блока;
  - каждый **n**-й элемент внутри блока (нумерация начинается с 1).

Программа обеспечивает ввод последовательности с клавиатуры, выполнение обработки и вывод результата.

---

## 2. Область применения

Данное программное средство может использоваться:

- для обработки цифровых сигналов;
- для уменьшения избыточности данных;
- для предварительной подготовки массивов измерений;
- в задачах анализа временных рядов.

---

## 3. Требования к программной среде

- Компилятор: **clang++** или **g++**
- Версия стандарта языка: **C++11** или выше
- Операционная система: macOS, Linux, Windows
- Средства ввода/вывода: консольный терминал

---

## 4. Алгоритм работы программы

1. Пользователь вводит:
   - количество элементов `N`;
   - `N` пар целых чисел `X` и `Y`, упорядоченных по `X`;
   - параметр `n`.

2. Программа разбивает входную последовательность на блоки, где значение `Y` не меняется.

3. Для каждого блока определяется его длина `len`.

4. Из блока выбираются элементы:
   - с локальным индексом `1`;
   - с локальным индексом `len`;
   - с локальным индексом, кратным `n`.

5. Отобранные элементы добавляются в результирующую последовательность.

6. На экран выводится:
   - исходная последовательность;
   - прореженная последовательность.
